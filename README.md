## To setup and run
1. Clone repository with all submodules: `git clone --recursive https://github.com/cyj0912/Construct3.git`
2. Run `download_dependency.cmd`
3. Setup `Data` folder
4. Point `ENGINE_ROOT` environment variable to the folder contains `Data`
5. `cmake Source`
